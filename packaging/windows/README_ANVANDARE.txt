flow for Windows
=====================

1. Kor "Installera flow.bat" om du packat upp zip-filen manuellt.
2. Appen installeras i din egen anvandarprofil och kraver inga admin-rattigheter.
3. Appen ansluter till den centrala flow-servern over internet.
4. Om en ny version finns far du en uppdateringsnotis i appen.

Tips:
- Om appen inte kan ansluta: prova "Forsok igen" eller "Oppna i webblasare".
- Du kan alltid starta appen igen via Start-menyn eller skrivbordsikonen.
